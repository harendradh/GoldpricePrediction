03 ▸3-tier agent memory · the determinism engine
💭
Tier 1
Conversational · ephemeral
Current chat session · last 10 turns · tool-call intermediate state · vanishes on page reload
Backed by: Redis or in-process dict
TTL: chat session lifetime
Scope: per (user, pipeline)
🧠
Tier 2 ★
Pipeline-scoped · durable
Prior decisions per pipeline · loaded as a "## Prior decisions" section before every new run · the reason a re-run produces byte-identical output
Backed by: df_studio.metadata.agent_memory (Delta)
Keys: defined per agent in YAML frontmatter
Scope: per (pipeline_id, agent, memory_key)
User-visible: yes — editable in UI
🌐
Tier 3
Org-wide patterns · learned
Vector search over all past pipelines · agent retrieves top-3 similar specs as background context · never enforced, only advisory
Backed by: df_studio.vs.past_specs
Embed: databricks-gte-large-en
Scope: workspace-wide (RBAC-filtered)
User-control: "use patterns" toggle per pipeline
